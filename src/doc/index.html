OpenFCST: The Open-source Fuel Cell Simulation Toolbox

Welcome to the OpenFCST Documentation index site. This site is used to access the user guide, application tutorials and class documentation within OpenFCST.

If you are a user, please start by reading the User Guide. Several tutorial examples on how to use OpenFCST can be found in the Tutorial Guide. If the folder is not available compile the examples by typing $make html in the terminal inside the examples folder.

In you are trying to develop your own fuel cell matehmatical model, the OpenFCST development team has developed a tutorial which documents how the application AppCathode was designed. The tutorial can be found here:
Application tutorial
Note that the tutorial does not cover the finite element theory. If you would like information on that part, please consult the deal.II tutorials

All the class in OpenFCST have been documented. The documentation for each class can be found in the following location:
Class documentation for OpenFCST

You can find further information regarding FCST at the project website

You can also join the openFCST Google group. This is the main discussion forum for openFCST users. Here they can ask queries, discuss code usage, suggest improvements, submit bugs etc.
